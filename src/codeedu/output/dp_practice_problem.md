编号: 001
类型: 动态规划
题干: 给定一个整数数组 nums 和一个整数目标值 target，返回可以使得 nums 中的元素相加得到目标值的所有不同组合的数目。每个数字可以被使用无限次。
示例:
- 示例 1:
  输入: nums = [1, 2, 3], target = 4
  输出: 7
  解释: 有 7 种不同的组合可以组成目标值 4：
  1. 1 + 1 + 1 + 1
  2. 1 + 1 + 2
  3. 1 + 2 + 1
  4. 2 + 1 + 1
  5. 2 + 2
  6. 1 + 3
  7. 3 + 1

- 示例 2:
  输入: nums = [2, 3, 5], target = 8
  输出: 5
  解释: 有 5 种不同的组合可以组成目标值 8：
  1. 2 + 2 + 2 + 2
  2. 2 + 3 + 3
  3. 3 + 5
  4. 3 + 2 + 3
  5. 2 + 5 + 2
